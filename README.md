# Unity Draw Graphs

Supports: 2019.1 + 

Usage: 
```
// string title, float value, Color color = default
DrawGraph.Add( "title", 33.3f );
```
Supports: Vector3, Vector2, Quaternion, float 

Open in application top level menu: `Window > Analysis > Graph`
